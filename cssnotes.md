# **CSS Notes**  

## **What is CSS?**  
> *CSS is a language used to add styling to a webpage that can be linked to an html file.*  

## **Examples**
``` h1{  
     float: left;    
     margin: 20px;    
     font-size: 60px;    
     font-family: fantasy;    
     font-weight: 50;     
} 
```      
```nav{  
    height: 100px;    
    background-color:rgb(54, 54, 54);    
    margin-bottom: 30px;    
    text-align: center;    
} 
```
